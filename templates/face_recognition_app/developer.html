About the Project
Face Recognition Attendance System

This project is a modern web-based attendance management system that uses facial recognition technology to automate the process of marking student attendance. It provides an efficient and accurate way to track student attendance while eliminating the need for manual record-keeping.

Key Features
  • Face Recognition Based Attendance
  • Real-time Processing
  • Student Management
  • Attendance Reports
  • Data Export Functionality
  • User-friendly Interface
